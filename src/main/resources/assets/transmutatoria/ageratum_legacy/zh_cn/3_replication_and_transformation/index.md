## 卷轴体系总览

### 复习

嬗变卷轴是炼金术士的核心工具，用于**炼金复制**和**炼金转化**。两种卷轴各有五个等级变体：

| 变体 | 关键原料 | 耐久   | 过期规则           |
|----|------|------|----------------|
| 嬗变 | 嬗变结晶 | 64   | 每天 12:00       |
| 地相 | 盐化基质 | 128  | 每天 12:00       |
| 月相 | 汞化基质 | 256  | 每 8 天（新月 0:00） |
| 日相 | 硫化基质 | 512  | 永不过期           |
| 无相 | 贤者之石 | 永不损坏 | 永不过期           |

**嬗变印记卷**用于炼金复制。将一件物品视为“模板”，在炼金锅中利用基材复制出一份新的。

**嬗变方程卷**用于炼金转化。将一种物品转化为另一种物品。

卷轴需要放入物品激活后才能使用。

### 一次性与可复用卷轴

卷轴分为**一次性**和**可复用**两种：

- **一次性卷轴**：成功产出物品后，催化剂槽位的卷轴会被消耗。
- **可复用卷轴**：成功产出物品后，只要卷轴仍有耐久，就可以再次放入转化原料和源质进行下一轮合成。

### 卷轴熵值

卷轴界面左下角的稳定度反映卷轴的熵值。卷轴的熵值越高，后续湮灭对卷轴的损耗越大。高熵值意味着你需要更多耐久度来完成全部槽位的湮灭。

## 执行合成的步骤

1. **投入卷轴**：将已激活的卷轴丢入炼金锅，它进入催化剂槽。
2. **投入输入**：将卷轴**左侧**槽位的物品丢入炼金锅，它进入输入槽。
3. **填充源质槽位**：从炼金术士储物盒中取出源质金属，填入 HUD 显示的每个源质输入槽。
4. **等待反应完成**：投入的源质数量满足卷轴要求后，反应自动开始。
5. **收取产物**：反应结束后，右键炼金锅取出产物。

部分炼金复制不需要输入（凭空复制），此时跳过第二步。

## 反应原理（重要）

- 每个**槽位上的源质**与**该槽位目标源质**按照投入顺序依次发生反应。基础规则见下表。某些**特殊槽位**会改变这一规则。
- 如果某个槽位投入的源质**与目标源质相同**，该槽位**触发湮灭**，并揭示该槽位的类型和目标源质。每次湮灭都会损耗卷轴。
- **只有当所有槽位全部触发湮灭**，且锅的极性在反应要求的极性范围内，输出槽才会产出最终物品。

| 情况        | 源质状态变化 | 锅极性变化 | 卷轴熵增       |
|-----------|--------|-------|------------|
| 源质与目标相生   | +1     | +1    | +1         |
| 源质克制目标    | +1     | -1    | +1         |
| 源质双倍克制目标  | +2     | -2    | +2         |
| 源质被目标克制   | -1     | +1    | +1         |
| 源质被目标双倍克制 | -2     | +2    | +2         |
| 源质与目标互相克制 | -1     | -1    | +1         |
| 源质与目标中性   | 0      | 0     | 0          |
| 源质与目标相同   | 湮灭     | 0     | 0 & 触发卷轴损耗 |

::: warning 注意
配方等级越高，槽位数量越多，需要投入的源质越多，全湮灭的难度也越大。
:::

## 炼金锅极性

炼金锅有极性属性，初始值为 0，范围为 -50 到 +50。破坏炼金锅方块后，炼金锅物品仍然保留极性属性。

炼金反应可能改变极性。部分炼金反应需要极性在范围内才能产出。

若极性超出范围：

- 极性 > 50：炼金锅变为**红石块**，所有物品掉落。
- 极性 < -50：炼金锅变为**残渣块**，所有物品掉落。

手持**贤者之石**右键炼金锅，可以将极性向 0 回调 1 点。

## 印记卷：炼金复制

### 原理

炼金复制将一件物品视为“模板”，通过源质反应在炼金锅中将其**复制出一份新的**。这需要存在对应的**炼金复制配方**。

### 激活

1. 使用印记卷，打开卷轴。
2. 在右侧槽位放入**你想复制的目标物品**。
3. 如果该物品存在炼金复制配方，卷轴激活。

激活后：
- 左侧槽位显示复制基材。部分复制配方不需要基材。
- 中部显示源质圆环，展示需要的源质数量（配方等级范围内的随机值）。

### 在炼金锅中使用

- 催化剂槽：激活后的印记卷轴。
- 转化输入槽：卷轴激活时显示的复制基材。
- 源质输入槽：按 HUD 显示的六边形槽位图投入对应源质。

## 方程卷：炼金转化

### 原理

炼金转化将一种物品转化为另一种物品。这需要存在对应的**炼金转化配方**。

### 激活

1. 使用方程卷，打开卷轴。
2. 在左侧槽位放入**你想被转化的原料物品**。
3. 如果该物品存在炼金转化配方，卷轴激活。

激活后：
- 右侧槽位显示转化后的产物。
- 中部显示源质圆环，展示需要的源质数量（配方等级范围内的随机值）。

### 在炼金锅中使用

- 催化剂槽：激活后的方程卷轴。
- 炼金输入槽：卷轴激活时放入的原料物品。
- 源质槽：按 HUD 显示的六边形槽位图投入对应源质。

::: warning 注意
配方等级越高，槽位数量越多，需要投入的源质越多，全湮灭的难度也越大。
:::

## 卷轴过期与突变

已激活的嬗变/地相/月相卷轴会随时间流逝而**过期**。卷轴过期后触发**突变**：

1. 槽位目标源质可能被随机替换。
2. 所有槽位的源质显示被隐藏。
3. 带方向槽位的箭头可能改变方向。

::: tip 提示
突变后所有源质再次变为未知，你需要重新通过反应试出每个槽位需要的源质。因此，卷轴被激活后应尽快使用，或使用日相/无相卷轴避免突变。
:::

## 熵值与卷轴耐久

### 熵值累积

每个槽位反应时，根据源质与槽位所需源质的关系累加熵值：

| 关系                   | 熵值增加 |
|----------------------|------|
| 双倍克制 / 被双倍克制         | +2   |
| 普通克制 / 被克制 / 互克 / 共生 | +1   |
| 中性/相同                | +0   |

### 耐久消耗

每个槽位上的源质湮灭时消耗卷轴耐久。熵值越高，后续湮灭对卷轴耐久造成的伤害越大。高熵值意味着你需要更多耐久度来完成全部槽位的湮灭。
