## 概述

掌握了源质基础后，你需要学会使用**嬗变卷轴**来复制和转化物品，使用**炼金术士储物盒**来管理你的源质金属，以及操作**混沌炼金锅**完成炼金合成。

## 制作卷轴

卷轴分为两种功能类型，各有五个等级变体。基础级的嬗变卷轴合成配方如下：

### 嬗变印记卷

*批注：什么雷霆中二表述，要这么写吗，不这么写怎么改*

用于**炼金复制**——凭空或利用基材复制物品。

| 材料     | 说明       |
|--------|----------|
| 任意源质金属 | 提供炼金亲和   |
| 嬗变结晶   | 核心催化     |
| 纸      | 承载符文     |
| 金粒     | 传导炼金能量   |
| 萤石粉    | 赋予“印记”之力 |

### 嬗变方程卷

用于**炼金转化**——将一种物品转化为另一种。

| 材料     | 说明       |
|--------|----------|
| 任意源质金属 | 提供炼金亲和   |
| 嬗变结晶   | 核心催化     |
| 纸      | 承载符文     |
| 金粒     | 传导炼金能量   |
| 红石粉    | 赋予“转化”之力 |

两种卷轴的区别仅在于萤石粉/红石粉的选择。

### 卷轴等级变体

基础嬗变卷轴可以通过添加更多材料升级：

| 变体 | 耐久   | 过期规则           |
|----|------|----------------|
| 嬗变 | 64   | 每天 12:00       |
| 地相 | 128  | 每天 12:00       |
| 月相 | 256  | 每 8 天（新月 0:00） |
| 日相 | 512  | 永不过期           |
| 无相 | 永不损坏 | 永不过期           |

::: tip 提示
高级卷轴需要更多资源，但提供了更好的耐久和更宽松的过期机制。详见[炼金复制与炼金转化](../3_replication_and_transformation/index.md)章节。
:::

## 激活卷轴

卷轴在刚制作出来时是**空白**的——需要放入一个目标物品来激活。

### 激活步骤

1. **手持卷轴右击**，打开卷轴。
2. 放入目标物品：
   - **印记卷**：放入你**想复制出的目标物**（右侧槽位）。
   - **方程卷**：放入你**想被转化的原料**（左侧槽位）。
3. 如果该物品存在对应的炼金规则，物品被卷轴**消耗**，卷轴被激活。

### 激活后的变化

- 卷轴显示配方另一端的物品预览。
- 中部显示**源质圆环**，标注所需源质的数量和类型。
- 刚激活后所有源质显示为 **`?`**（未知），需要在炼金锅中通过反应逐一揭示。

::: warning 注意
激活时放入的物品会被**消耗**。请预先确认你放入的是正确的物品。
:::

## 炼金术士储物盒

炼金术士储物盒是管理源质金属的核心工具。它同时是**物品**和**方块**。

### 功能

- 提供 **12 个槽位**，每个槽位绑定一种源质金属。每种源质金属可存储一组（64个）。
- **自动收纳**：捡起源质金属时，物品栏中的储物盒会自动吸入匹配的源质金属。
- **快速投料**：手持储物盒对准炼金锅，可以快速将选中的源质金属投入锅的源质槽。

### 物品形态

储物盒可以作为便携式源质容器使用：

- **手持储物盒右击**：打开储物盒。

### 方块形态

储物盒可以放置在地上，作为方块和容器使用：

- **右击方块**：打开储物盒。
- **Shift + 右击方块**：将方块收回为物品形态。
- 支持自动化 IO。

### 四种状态变体

储物盒有四种变体，只能收纳**对应状态**的源质金属：

| 储物盒       | 收纳范围   |
|-----------|--------|
| 炼金术士储物盒   | 常态源质金属 |
| 黑化炼金术士储物盒 | 黑化源质金属 |
| 白化炼金术士储物盒 | 白化源质金属 |
| 黄化炼金术士储物盒 | 黄化源质金属 |

## 混沌炼金锅

混沌炼金锅是完成炼金反应的核心工作站。看向它时会显示 HUD 界面。

### 槽位

混沌炼金锅有四类物品槽位：

- **催化剂槽**：存放反应的催化剂，催化剂决定反应类型。
- **输入槽**：存放反应的输入物品。
- **源质槽**：存放反应所需的源质金属和产出的源质金属。
- **输出槽**：存放反应的输出物品。

### HUD 布局

| 位置  | 显示内容                         |
|-----|------------------------------|
| 最左侧 | 卷轴当前耐久，紫色部分 = 预计消耗           |
| 左侧  | 上：催化剂槽与输入槽 · 中：炼金进度 · 下：输出槽  |
| 中央  | 源质槽位图                        |
| 中上  | 源质轮盘（手持炼金术士储物盒时显示）           |
| 右上  | 沙漏：卷轴过期倒计时 · 指针：当前极性与要求的极性范围 |

### 源质轮盘操作

手持储物盒看向炼金锅时，屏幕上方显示 **12 槽环形轮盘**：

- **中间**是当前选中的源质。
- **滚动鼠标滚轮**切换选中的源质金属。
- 按住 Alt 展开源质轮盘，简单显示源质金属之间的克制关系。

### 投料与槽位选择

催化剂和炼金输入的投料方式是直接**丢入锅内**，它们会自动进入对应槽位。只有先投入催化剂，才能投入输入物品/源质金属。

源质金属有两种投料方式：
- 手持储物盒并**对着炼金锅右击**，投入当前选中的源质金属。
- 直接将源质金属**丢入锅内**。

源质金属会进入源质槽位。源质槽位图上的**选择环**指示当前操作的源质槽位，**Shift + 滚轮**可以切换。

只有当前操作的源质槽位为空时，才能接收投入的源质金属。成功接收后，当前操作槽位将自动移至下一槽位。

### 反应触发条件

反应开始需要满足以下全部条件：

1. 当前没有正在进行的反应。
2. 没有未取走的产物。
3. 催化剂已放入。
4. 根据催化剂类型，炼金输入和（或）源质输入已就位。
5. 锅内有足够的水。

满足条件后反应自动开始。

### 产物取出

反应结束后，右击炼金锅取出源质槽和输出槽的产物。

反应开始前或反应途中，空手 Shift + 右击炼金锅将依次取出源质槽、输入槽和催化剂槽的物品。反应途中取出将终止反应。

### 自动化 IO

| 槽位   | 类型              | 面              |
|------|-----------------|----------------|
| 水槽   | 仅输入             | 所有面            |
| 催化剂槽 | 仅输入             | 顶面和底面          |
| 输入槽  | 仅输入             | 顶面和底面          |
| 源质槽  | 输入（反应前）/输出（反应后） | 侧面（输入）/所有面（输出） |
| 输出槽  | 仅输出             | 所有面            |










## 炼金合成

*批注：这内容跟第三章重复了吧？*

*批注：炼金复制和炼金转化作为最核心的玩法单拉一章重点阐述？*

激活卷轴后，将其投入混沌炼金锅的催化剂栏位。HUD 会显示卷轴的源质槽位图——这是一组六边形连通的槽位，每个槽位标记着它需要的源质金属类型（初始为未知，每解锁源质会显示）。

### 执行合成的步骤

1. **投入卷轴**：将已激活的卷轴丢入炼金锅，它进入催化剂槽位。
2. **放入炼金原料**：印记卷轴需要放入**你想要复制的那个物品**；方程卷轴需要放入**你想被转化的那个物品**。放入的物品必须与卷轴激活时所用的物品类型一致。
3. **填充源质槽位**：从炼金术士储物盒中取出源质金属，填入 HUD 显示的每个源质输入槽。
4. **等待反应完成**：投入的源质数量满足卷轴要求后，反应自动开始。
5. **收取产物**：反应结束后，产物出现在炼金输出槽。右键炼金锅即可取出。

### 反应的内部过程

反应按你投放源质的顺序逐个执行槽位：

- 每个槽位根据**投入的源质**与**该槽位目标源质**的关系计算状态变化。基础规则与源质反应相同（克制方活化、被克制方劣化），但每个槽位的**特殊类型**会改变这一规则。
- 反应过程中，输入源质会被消耗并产生对应状态变化的输出源质，写入输出槽。
- 如果某个槽位投入的源质**与目标源质相同**，该槽位**触发损毁**（湮灭），并揭示该槽位的类型和目标源质。每次触发损毁都会消耗卷轴的耐久度，消耗量 = `1 + 当前熵值 / 4`。
- **只有当所有槽位全部触发损毁**，且锅的极性在卷轴配方要求的极性范围内，且转化输入仍然匹配卷轴要求时，转化输出槽才会产出最终物品。
- 无论是否成功产出，反应结束后所有输入源质都会被清空。

### 一次性与可复用卷轴

*批注：这内容应该放在第三章吧？*

卷轴分为**一次性**和**可复用**两种：

- **一次性卷轴**：成功产出物品后，催化剂槽位的卷轴会被消耗。
- **可复用卷轴**：成功产出物品后，只要卷轴仍有耐久，就可以再次放入转化原料和源质进行下一轮合成。

### 极性管理

*批注：这内容应该放在第三章吧？*

每次炼金合成反应都会改变锅的极性。若极性超出 ±50：

- 极性 > 50：锅变为**红石块**，所有物品掉落。
- 极性 < -50：锅变为**炼金渣块**，所有物品掉落。

手持**贤者之石**右键炼金锅，可以将极性向 0 回调 1 点，用于微调。

### 观察 HUD 信息

*批注：重复了*

- **最左侧金属管**：显示卷轴当前耐久度和熵值累积。紫色部分代表合成整件物品预计消耗的耐久。
- **左边竖条**：上中下分别显示催化剂、炼金输入、炼金输出。
- **中间六边形区域**：显示卷轴的源质槽位图。已揭示的槽位会显示源质图标。
- **右上沙漏圆盘**：显示卷轴过期倒计时。外圈指示极性状态。
