## 概述

炼金的第一步是获取**嬗变结晶**——它是整个炼金术的起点材料。通过嬗变结晶制作混沌炼金锅，再利用末影之眼将结晶分解为**源质金属**，你便踏入了源质的世界。

## 嬗变结晶制取

### 材料准备

| 材料   | 数量     |
|------|--------|
| 红石粉  | 1      |
| 火药   | 1      |
| 萤石粉  | 1      |
| 绿宝石块 | 1（不消耗） |
| 炼药锅  | 1（装水）  |

### 步骤

1. 在地上放置**绿宝石块**。
2. 在绿宝石块上方放置**炼药锅**，用水桶装满水。
3. 将红石粉、火药、萤石粉**丢入**炼药锅中。
4. 三者接触水面后转化为**嬗变结晶**。

### 混沌炼金锅制作

用嬗变结晶和两个金锭与炼药锅在工作台合成**混沌炼金锅**：

*批注：配方显示是坏的吗*

| 材料   | 位置  |
|------|-----|
| 金锭   | 左、右 |
| 嬗变结晶 | 中间  |
| 炼药锅  | 下方  |

::: tip 提示
嬗变结晶的制取不消耗绿宝石块和炼药锅，可以反复使用同一个装置批量生产。
:::

## 炼金分解

这是获取源质金属的唯一初始途径——通过末影之眼“观测”嬗变结晶，揭示其内部的源质本质。

**操作步骤：**

1. 将混沌炼金锅放在地上，用水桶装入水。
2. 投入**末影之眼**。末影之眼是催化剂，**不会被消耗**。
3. 投入**嬗变结晶**。
4. 反应自动开始。

**产出：**

| 概率  | 产物                 |
|-----|--------------------|
| 99% | 前 11 种常态源质金属中的随机一种 |
| 1%  | 常态**无相源金**         |

每次反应消耗 **15 mB 水**。建议多准备嬗变结晶，批量进行分解以获取多样化的源质金属。

::: warning 注意
如果锅内水量不足，反应不会启动。确保锅里有足够的水。
:::

## 源质金属

源质金属共有 **12 种**，分属四大元素柱，每柱三种。

### 四柱十二金

| 四柱                  | 三种源质                                                     |
|---------------------|----------------------------------------------------------|
| 星火 · Astral Fire    | 蚀日金 · Eclipsium、泣月银 · Lunargent、灾星钛 · Astrotite          |
| 渊水 · Abyssal Water  | 渊海钢 · Abyssion、幻魂汞 · Animercury、冥沼铅 · Necroplumb         |
| 腐土 · Decaying Earth | 棘血铜 · Sanguibronze、鸩林锡 · Venotite、骸骨锑 · Ossantimony      |
| 劫风 · Tempest Air    | 怒雷锌 · Fulgurzinc、凝时铂 · Chronoplatinum、无相源金 · Pandemonium |

### 相克相生关系

源质金属之间存在三层关系：**柱间基础克制**、**元素克制**和**共生**，三者叠加形成最终的反应表。

#### 柱间基础克制

| 克制方 | 被克制方 |
|-----|------|
| 星火  | 劫风   |
| 劫风  | 腐土   |
| 腐土  | 渊水   |
| 渊水  | 星火   |

克制形成一个循环：星火 → 劫风 → 腐土 → 渊水 → 星火。

#### 元素克制关系

在柱间克制之上，特定源质金属之间还有额外的克制关系：

| 克制方 | 被克制方    |
|-----|---------|
| 冥沼铅 | 蚀日金、骸骨锑 |
| 棘血铜 | 泣月银     |
| 渊海钢 | 幻魂汞     |
| 鸩林锡 | 棘血铜     |
| 骸骨锑 | 鸩林锡     |
| 凝时铂 | 灾星钛     |

#### 元素共生关系

部分源质金属之间存在共生关系：

| 金属 A | 金属 B |
|------|------|
| 泣月银  | 鸩林锡  |
| 渊海钢  | 怒雷锌  |
| 棘血铜  | 骸骨锑  |

#### 无相源金

无相源金是极其特殊的例外：

- 炼金分解中获取概率仅 **1%**。
- **不受四柱关系影响**——不被任何源质克制。
- **克制所有其他 11 种源质**。
- 是多种高级配方和终极炼金的关键材料。

## 翠绿石板

由 1 个嬗变结晶和 4 个绿宝石在工作台合成。

翠绿石板是一个交互式参考工具——使用后可以查看任意两种源质金属之间的完整克制关系，对后续的炼金非常有帮助。

## 源质状态与反应

### 四种状态

所有的源质金属都有四种状态：

| 状态值   | 名称              | 说明           |
|-------|-----------------|--------------|
| < -1  | 残渣              | 过度劣化产物，不可再用  |
| -1    | 黑化 · Nigredo    | 劣化一阶         |
| 0     | 常态              | 基础形态         |
| +1    | 白化 · Albedo     | 活化一阶         |
| +2    | 黄化 · Citrinitas | 活化二阶         |
| \> +2 | 红石粉             | 过度活化产物，还原为红石 |

::: tip 提示
源质金属的状态不影响本质，不影响相生相克关系。
:::

### 反应规则

两个源质金属发生反应时，根据两者的关系，双方状态各自变化：

| 关系          | A 状态变化 | B 状态变化 |
|-------------|--------|--------|
| A 克制 B      | +1     | -1     |
| A 双倍克制 B    | +2     | -2     |
| B 克制 A      | -1     | +1     |
| B 双倍克制 A    | -2     | +2     |
| A 与 B 互相克制  | -1     | -1     |
| A 与 B 共生    | +1     | +1     |
| A 与 B 中性或相同 | 0      | 0      |

- **克制方活化**：状态提升（向白化/黄化方向）。
- **被克制方劣化**：状态降低（向黑化/炼金渣方向）。
- 过度活化得到红石粉，过度劣化得到残渣。

### 实例

以常态**灾星钛**和常态**渊海钢**反应为例。灾星钛属于星火柱，渊海钢属于渊水柱，渊水克制星火 → 渊海钢克制灾星钛。

结果：灾星钛 **-1** → 黑化灾星钛；渊海钢 **+1** → 白化渊海钢。
