## 伟大之作 · Magnum Opus

伟大之作是炼金术的终极目标——**贤者之石**的炼制。这需要经历四个阶段（黑化、白化、黄化、赤化），每个阶段都要求特定的极性条件和源质槽位数量。整个炼制过程是对炼金术士全部技能的终极考验。

## 炼金宇宙论

在开始伟大之作之前，你需要理解三基质与四阶段的概念：

### 三基质

古代炼金术认为万物由三种基质构成：

| 基质          | 象征       | 对应物品 |
|-------------|----------|------|
| 盐 · Sal     | 肉体、固定、形体 | 盐基质  |
| 汞 · Mercury | 灵魂、流动、连接 | 汞基质  |
| 硫 · Sulfur  | 精神、活化、转化 | 硫基质  |

### 三基质的炼制

三种基质通过炼金转化依次制取，极性要求逐级升高：

| 基质  | 原料  | 等级 | 极性要求   |
|-----|-----|----|--------|
| 盐基质 | 沙子  | 4  | ≤ 0    |
| 汞基质 | 盐基质 | 8  | ≥ 20   |
| 硫基质 | 汞基质 | 13 | -1 ~ 1 |

::: tip 提示
硫基质要求极性在 -1 到 1 之间——这是一个极窄的窗口。你需要精确调整锅的极性才能满足条件。
:::

### 第一原质

**第一原质**是所有物质的原始基底。用 1 个嬗变结晶和 4 个圆石在工作台合成：

```
 圆石
圆石 结晶 圆石
 圆石
```

第一原质是伟大之作的**起点原料**。

## 四个阶段

### 第一阶段：黑化 · Nigredo

> *"下如同上，上如同下"*

黑化是伟大之作的起点——原初物质在黑暗中被分解、腐化、归为混沌。

| 条件   | 值             |
|------|---------------|
| 转化原料 | 第一原质          |
| 产物   | 黑化质料          |
| 等级   | 3             |
| 极性要求 | **-50 ~ -10** |

这是最简单的阶段——槽位少，但要求锅极性为负。

### 第二阶段：白化 · Albedo

> *"从地养护，从风孕育"*

白化是净化的阶段——黑暗被洗净，物质被提纯为银白之光。

| 条件   | 值           |
|------|-------------|
| 转化原料 | 黑化质料        |
| 产物   | 白化质料        |
| 等级   | 6           |
| 极性要求 | **20 ~ 50** |

极性反转——需要在黑化阶段后大幅提升极性至正值。

### 第三阶段：黄化 · Citrinitas

> *"太阳为父，月亮为母"*

黄化是觉醒的阶段——银白之光被太阳染成金黄，物质获得转化之力。

| 条件   | 值           |
|------|-------------|
| 转化原料 | 白化质料        |
| 产物   | 黄化质料        |
| 等级   | 12          |
| 极性要求 | **40 ~ 50** |

源质数、槽位数翻倍至 12，特殊槽位开始出现，极性窗口窄且高。

### 第四阶段：赤化 · Rubedo

> *"这便是万力之力和无穷之力"*

赤化是完成的阶段——金色被点燃为赤红，贤者之石的雏形在此诞生。这是**一次性**转化。

| 条件   | 值           |
|------|-------------|
| 转化原料 | 黄化质料        |
| 产物   | 赤化质料        |
| 等级   | **24**      |
| 极性要求 | **49 ~ 50** |
| 一次性  | **是**       |

这是整个伟大之作中最困难的一步：
- 有 24 个槽位，特殊槽位概率约 50%。
- 极性必须精确落在 49~50 的极限窗口——仅 2 点容差。
- 卷轴是一次性的——第二次合成必须重新尝试配方。

::: warning 准备建议
在开始赤化之前：
1. 使用**月相卷轴或日相卷轴**——24 槽位需要大量耐久。
2. 准备足够的源质金属储备——24 个槽位需要 24 个匹配的源质。
3. 确保锅极性稳定在 49~50。
4. 仔细研究槽位图，规划投放顺序。
:::

### 最终步骤：煅烧

获得赤化质料后，将其放入**熔炉**煅烧，即可得到贤者之石。

## 贤者之石的权能

贤者之石是炼金术的至高成就。持有它即可获得以下能力：

### 生命守护

- **持续再生**：主手或副手持有时，持续获得生命恢复效果。
- **饱和滋养**：每秒获得短暂饱和效果，免除进食需求。
- **免死庇佑**：当快捷栏中存在未冷却的贤者之石时，任何致命伤害会被抵消——生命值锁定在 1 点，该贤者之石进入 **60 秒冷却**。冷却期间该石头暂时失去免死能力。

::: tip 提示
只有快捷栏内有贤者之石，免死效果才生效，背包中的石头不生效。多颗贤者之石可以轮流使用。
:::

### 治愈腐化

手持贤者之石右键**僵尸村民**，直接完成其转化过程，无需等待原版的虚弱药水 + 金苹果流程。

### 极性调节

手持贤者之石右键**混沌炼金锅**，将锅的极性向 0 回调 **1 点**。

### 混沌分解

贤者之石可作为催化剂投入炼金锅，将可被炼金复制的物品**逆向分解**为源质金属。

1. 将贤者之石投入催化剂槽。
2. 投入能被被印记卷轴复制出来的物品。
3. 物品的炼金复制等级越高，反应越慢。
4. 炼金输入被分解，产出与配方等级成正比数量的常态源质金属。

这是**大量获取源质金属**的有效方式——将不再需要的、有复制配方的物品分解回源质。
