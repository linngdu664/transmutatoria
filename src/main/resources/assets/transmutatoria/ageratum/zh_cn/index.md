# 转化秘典：混沌炼金术

![](transmutatoria:item/liber_transmutationis_ars_transmutatoria.png)

> 万物并非被创造或毁灭，只是在形态、纯度与秩序之间迁流。炼金术士所做的，是听见这种迁流，然后付出相称的代价。

这不是一本要求你先背完规则的辞典。请沿着左侧章节依次实践：先取得嬗变结晶与源质，再学会读懂反应，最后完成从第一原质到贤者之石的“大工”。

## 第一次游玩的最短路线

1. 阅读[第一簇火花](01_first_steps/index)，制得嬗变结晶与嬗变之锅。
2. 用末影之眼分解结晶，取得第一枚源质金属。
3. 阅读[十二源质](02_essence/index)，用翠绿石板认识相生、克制与四种状态。
4. 制作储物盒和一张卷轴，完成第一次[炼金转化](03_scrolls/04_transformation)。
5. 用基质与高阶卷轴稳定更复杂的炼金，最终进入[伟大工程](04_great_work/index)。

::: tip
如果眼前的锅没有按预期工作，先查四件事：**有水、催化剂正确、原料正确、源质槽位已填满**。仍有问题时打开[故障排查](06_reference/02_troubleshooting)。
:::

## 这门炼金术在讲什么

世界由四柱承载：**星火、渊水、腐土、劫风**。每柱分化出三种源质金属，共十二种。它们彼此滋生、压制或湮灭；每次反应同时改变金属的形态与锅的**源质极性（EP）**。

你真正要掌握的不是一张固定配方表，而是三个判断：

- 我要让投入的源质发生什么关系？
- 反应会把锅的 EP 推向哪里？
- 我愿意用多少卷轴稳定性来换取这次转化？
