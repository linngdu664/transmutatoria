## 概述

炼金术的第一步是获取**嬗变结晶**——它是整个模组的起点材料。通过嬗变结晶制作混沌炼金锅，再利用末影之眼将结晶分解为**源质金属**，你便踏入了源质的世界。

## 嬗变结晶制取

### 材料准备

| 材料 | 数量 |
|---|---|
| 红石粉 | 1 |
| 火药 | 1 |
| 萤石粉 | 1 |
| 绿宝石块 | 1（不可消耗） |
| 炼药锅 | 1（装水） |

### 步骤

1. 在地上放置**绿宝石块**。
2. 在绿宝石块上方放置**炼药锅**，用水桶装满水。
3. 将红石粉、火药、萤石粉**丢入**炼药锅中（按 Q 键丢弃物品到锅中）。
4. 三者接触水面后转化为**嬗变结晶**。

### 混沌炼金锅制作

用嬗变结晶和两个金锭与炼药锅在工作台合成**混沌炼金锅**：

| 材料 | 位置 |
|---|---|
| 金锭 | 左、右 |
| 嬗变结晶 | 中间 |
| 炼药锅 | 下方 |

::: tip 提示
嬗变结晶的制取不消耗绿宝石块和炼药锅，可以反复使用同一个装置批量生产。
:::

## 炼金分解

### 催化剂：末影之眼

这是获取源质金属的唯一初始途径——通过末影之眼"观测"嬗变结晶，揭示其内部的源质本质。

**操作步骤：**

1. 将混沌炼金锅放在地上，用水桶装入水（最多 1000 mB）。
2. 将**末影之眼**投入催化剂槽位。
3. 将**嬗变结晶**投入转化输入槽位。
4. 反应自动开始，耗时 **10 tick**（0.5 秒）。

**产出：**

| 概率 | 产物 |
|---|---|
| 99% | 前 11 种常态源质金属中的随机一种 |
| 1% | **无相源金**（Pandemonium） |

每次反应消耗 **20 mB 水**。建议多准备嬗变结晶和末影之眼，批量进行分解以获取多样化的源质金属。

::: warning 注意
如果锅内水量不足（< 20 mB），反应不会启动。确保锅里有足够的水。
:::

## 源质金属

源质金属共有 **12 种**，分属四大元素柱，每柱三种。

### 四柱十二金

| 四柱 | 代号 | 三种源质 |
|---|---|---|
| 星火 · Astral Fire | A/B/C | 蚀日金 · Eclipsium、泣月银 · Lunargent、灾星钛 · Astrotite |
| 渊水 · Abyssal Water | D/E/F | 渊海钢 · Abyssion、幻魂汞 · Animercury、冥沼铅 · Necroplumb |
| 腐土 · Decaying Earth | G/H/I | 棘血铜 · Sanguibronze、鸩林锡 · Venotite、骸骨锑 · Ossantimony |
| 劫风 · Tempest Air | J/K/L | 怒雷锌 · Fulgurzinc、凝时铂 · Chronoplatinum、无相源金 · Pandemonium |

### 克制关系

源质之间存在两层克制关系：**柱间基础克制**和**金属间元素克制**，两者叠加形成最终的反应表。

#### 柱间基础克制

| 克制方 | 被克制方 |
|---|---|
| 星火 | 劫风 |
| 劫风 | 腐土 |
| 腐土 | 渊水 |
| 渊水 | 星火 |

克制形成一个循环：星火 → 劫风 → 腐土 → 渊水 → 星火。

#### 元素克制关系

在柱间克制之上，特定金属之间还有额外的克制关系：

| 克制方 | 被克制方 |
|---|---|
| 冥沼铅（F） | 蚀日金（A）、骸骨锑（I） |
| 棘血铜（G） | 泣月银（B） |
| 渊海钢（D） | 幻魂汞（E） |
| 鸩林锡（H） | 棘血铜（G） |
| 骸骨锑（I） | 鸩林锡（H） |
| 凝时铂（K） | 灾星钛（C） |

#### 元素共生关系

部分金属之间存在共生关系——反应时双方状态都会提升：

| 金属 A | 金属 B |
|---|---|
| 泣月银（B） | 鸩林锡（H） |
| 渊海钢（D） | 怒雷锌（J） |
| 棘血铜（G） | 骸骨锑（I） |

#### 无相源金

无相源金（Pandemonium，L）是极其特殊的例外：

- 炼金分解中获取概率仅 **1%**。
- **不受四柱关系影响**——不被任何源质克制。
- **克制所有其他 11 种源质**。
- 是多种高级配方和终极炼金的关键材料。

## 翠绿石板

由 1 个嬗变结晶和 4 个绿宝石在工作台合成。

翠绿石板是一个交互式参考工具——右键打开后可以查看任意两种源质金属之间的完整克制关系。对于规划反应序列和管理极性非常有帮助。

::: tip 提示
在炼金锅前手持翠绿石板，可以快速查阅关系表，无需记忆全部 66 对关系。
:::

## 源质状态与反应

### 四种状态

| 状态值 | 名称 | 前缀 | 说明 |
|---|---|---|---|
| < -1 | 炼金渣 | — | 过度劣化产物，不可再用 |
| -1 | 黑化 · Nigredo | `nigredo_*` | 劣化一阶 |
| 0 | 常态 | 无前缀 | 基础形态 |
| 1 | 白化 · Albedo | `albedo_*` | 活化一阶 |
| 2 | 黄化 · Citrinitas | `citrinitas_*` | 活化二阶 |
| > 2 | 红石粉 | — | 过度活化产物，还原为原版红石 |

### 反应规则

两个源质金属发生反应时，根据两者的关系，双方状态各自变化：

| 关系 | self（槽位0） | other（槽位1） | 极性变化 |
|---|---|---|---|
| 共生 | +1 | +1 | +2 |
| 克制 | +1 | -1 | 0 |
| 双倍克制 | +2 | -2 | 0 |
| 被克制 | -1 | +1 | 0 |
| 双倍被克制 | -2 | +2 | 0 |
| 互克 | -1 | -1 | -2 |
| 中性 | 0 | 0 | 0 |
| 相同 | 0 | 0 | 0 |

- **克制方活化**：状态提升（向白化/黄化方向）。
- **被克制方劣化**：状态降低（向黑化/炼金渣方向）。
- 状态超出 -1 边界变为炼金渣，超出 +2 边界变为红石粉。

### 实例

以常态**灾星钛（C）**和常态**渊海钢（D）**反应为例。灾星钛属于星火柱，渊海钢属于渊水柱，渊水克制星火 → 渊海钢克制灾星钛。

- 设灾星钛在槽位 0（self），渊海钢在槽位 1（other）。
- 关系为**被克制**：self = -1，other = +1。
- 结果：灾星钛 **-1** → 黑化灾星钛；渊海钢 **+1** → 白化渊海钢。
- 极性变化：0。

::: warning 注意
**投入顺序决定谁是 self、谁是 other**。如果把两者位置互换——渊海钢放槽位 0，灾星钛放槽位 1——则关系变成**克制**而非被克制，结果完全相反。
:::
