# 故障排查

## 物品落进锅里，却没有被吸入

- 检查前一个位置是否已占用：锅按催化剂 → 原料 → 源质识别。
- 检查催化剂是否匹配当前反应。
- 卷轴必须先激活；未激活卷轴不能作为锅的催化剂。
- 卷轴要求的普通输入必须与激活时记录的物品一致。
- 确保物品确实落在锅内部，不是锅沿或旁边。

## 槽位填不进去

- 锅至少需要 20 mB 水。
- 先投入催化剂，以及配方要求的普通输入。
- 当前槽位可能已有源质；潜行滚轮换到空槽。
- 源质融合需要不同的完整克制集合，重复品不能替代缺项。

## 填满后没有开始

- 卷轴槽位必须全部填满。
- 源质反应必须恰好投入两枚源质。
- 源质融合必须覆盖催化源质克制及双倍克制的全部种类。
- 水量不足会让已布置反应等待，补水后再观察。

## 全部源质都消失了，却没有目标产物

这通常说明槽位都正确湮灭了，但锅的最终 EP 不在配方区间。取回卷轴，调整 EP 后重试。也要确认规则是否要求高阶卷轴或属于一次性规则。

## 返回了一圈状态不同的源质

这不是系统故障，而是卷轴侦察结果：对应位置没有投入正确的同种源质。根据活化、劣化或不变推断关系；已湮灭的位置会在卷轴上留下记录。

## 卷轴答案突然变了

基础与地相卷轴会周期性突变，月相卷轴会在新月重排。查看卷轴到期提示；需要长期固定序列时升级到日相或无相。

## 锅变成红石块或残渣块

EP 超出了 +50 或 -50。锅和内容物已经成为反应的一部分，无法从原方块取回。下次在高风险操作前把 EP 拉回中段，或使用专门的调极性锅。

## 高级配方始终失败

- 等级大于 8 时至少使用地相卷轴。
- 特殊槽位可能伪装；先通过湮灭揭示。
- 把即时与延迟效果分开计算。
- 扩散、共振和反转会让最终状态不同于基础关系表。

::: tip
排错顺序固定为：**水 → 催化剂 → 普通输入 → 槽位数量与顺序 → EP → 卷轴阶位 → 特殊槽位**。这样最快，也最不容易为了一个简单缺水问题重算整张关系图。
:::
