# 反应速查

## 四柱

**渊水 → 星火 → 劫风 → 腐土 → 渊水**

无相源金 L 单向克制 A 至 K，且不受四柱克制。

## 两源质反应

催化剂为嬗变结晶。表中 A 是先投入者，B 是后投入者。

| A 相对 B 的关系 | A 状态 | B 状态 | 锅 EP |
|---|---:|---:|---:|
| 相生 | +1 | +1 | -2 |
| A 克制 B | +1 | -1 | 0 |
| A 双倍克制 B | +2 | -2 | 0 |
| A 被 B 克制 | -1 | +1 | 0 |
| A 被 B 双倍克制 | -2 | +2 | 0 |
| 互相克制 | -1 | -1 | +2 |
| 无关 | 0 | 0 | 0 |

## 卷轴槽位反应

“投入源质”与“卷轴在该槽记录的源质”逐一比较。

| 关系 | 投入源质状态 | 锅 EP | 熵增 |
|---|---:|---:|---:|
| 相生 | +1 | +1 | +1 |
| 投入者克制记录者 | +1 | -1 | +1 |
| 投入者双倍克制记录者 | +2 | -2 | +2 |
| 投入者被记录者克制 | -1 | +1 | +1 |
| 投入者被记录者双倍克制 | -2 | +2 | +2 |
| 互相克制 | -1 | -1 | +1 |
| 无关 | 0 | 0 | 0 |
| 相同 | 湮灭 | 0 | 触发卷轴损耗 |

## 状态边界

| 数值 | 结果 |
|---:|---|
| 小于 -1 | 残渣 |
| -1 | 黑化 |
| 0 | 普通 |
| +1 | 白化 |
| +2 | 黄化 |
| 大于 +2 | 红石粉 |

## 取物操作

- 空手右击：取出完成的产物与反应后源质。
- 潜行 + 空手右击：依次撤回未反应源质、原料、催化剂。
- 储物盒滚轮：选择源质。
- 储物盒右击锅：注入当前源质。
- 看向锅潜行 + 滚轮：选择炼金槽位。
