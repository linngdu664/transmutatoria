# 卷轴炼金

源质实验改变的是源质；卷轴炼金改变的是**其他物质**。卷轴先从一个物品读取目标，再生成一组有顺序的炼金槽位。你要逐槽投入源质，让它们与卷轴记录的源质发生湮灭。

## 两类卷轴

| 卷轴 | 你放入卷轴的物品 | 它回答的问题 |
|---|---|---|
| 印记卷 | 想得到的目标物 | “怎样复制这个？” |
| 方程卷 | 想改变的原料 | “它能转化成什么？” |

两类卷轴的锅边操作相同；区别只在激活时记录的是结果还是输入。
