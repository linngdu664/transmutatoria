# 卷轴阶位与稳定性

基础嬗变卷轴适合学习，但会随时间突变槽位，也很难承受长序列。高阶卷轴不是单纯“耐久更高”，而是逐步摆脱物质世界的无常。

| 阶位 | 核心材料 | 特性 |
|---|---|---|
| 嬗变 | 任意源质 + 嬗变结晶 | 耐久 32；序列会周期性突变 |
| 地相 | 灾星钛 + 盐化基质 | 耐久 64；可进行等级大于 8 的高级炼金 |
| 月相 | 泣月银 + 汞化基质 | 耐久 96；只在新月周期重新排列 |
| 日相 | 蚀日金 + 硫化基质 | 耐久 128；不会重新排列 |
| 无相 | 无相源金 + 贤者之石 + 日相卷轴 | 不损耗耐久，不会重新排列 |

印记卷再加入**纸、金粒与萤石粉**；方程卷再加入**纸、金粒与红石粉**。贤者之石参与制作无相卷轴时不被消耗。

## 突变意味着什么

基础与地相卷轴会在时间周期到达时随机改写部分槽位，并隐藏新内容。月相卷轴只在新月更替，日相与无相卷轴则固定序列。

::: tip
卷轴显示即将过期时，不要启动一场长反应。先等它突变、重新侦察，或升级卷轴。最昂贵的损失往往不是卷轴本身，而是一套已经验证到一半的答案。
:::

## 选择建议

- 等级 1–8、目标便宜：嬗变卷轴。
- 等级 9 以上：至少地相卷轴。
- 需要长期保留同一套配方：月相或日相。
- 终局自动化与珍贵一次性规则：无相卷轴。

高阶炼金还会生成特殊槽位；参见[高阶炼金](../05_advanced/index)。
