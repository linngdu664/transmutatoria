# 炼金转化

炼金转化从“输入”寻找唯一结果：向方程卷放入原料，卷轴会记录它可抵达的目标以及完成变化所需的源质序列。

## 一次完整转化

1. 右击未激活方程卷，把**要改变的原料**放入卷轴。
2. 阅读卷轴显示的结果、等级和 EP 要求。
3. 把激活方程卷投入嬗变之锅。
4. 再投入与卷轴记录相同的原料。
5. 按序填满源质槽位。
6. 全部湮灭且 EP 合格后，取出目标物。

印记卷回答“怎样得到它”，方程卷回答“它能变成什么”。除此之外，试探未知槽位、控制 EP 和取回产物的方式完全相同。

## 基础转化练习

建议先选择等级较低、材料易得的规则，例如圆石或铁锭向金的转化。第一次练习的目标不是盈利，而是完整经历：**激活 → 读卷 → 投原料 → 填槽 → 观察返还 → 修正 → 全湮灭**。

## 三种基质与四阶段

方程卷也是伟大工程的工具。盐化、汞化、硫化基质，以及黑化、白化、黄化、赤化质料，都要求特定等级与 EP。详细路线见[三基质](../04_great_work/01_matrices)和[四阶段](../04_great_work/02_four_stages)。

::: warning
配方记录的输入物与实际投入物必须一致。卷轴能打开、源质槽能显示，不代表你随后扔进锅的原料可以随意替换。
:::
