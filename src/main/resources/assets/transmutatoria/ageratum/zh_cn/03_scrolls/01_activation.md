# 激活与解读卷轴

## 制作基础卷轴

- **嬗变印记卷**：任意普通源质金属 + 嬗变结晶 + 纸 + 金粒 + 萤石粉。
- **嬗变方程卷**：任意普通源质金属 + 嬗变结晶 + 纸 + 金粒 + 红石粉。

印记用萤石表达“把目标照回世间”，方程用红石表达“让变化沿路径发生”。

## 激活

1. 手持未激活卷轴右击，打开卷轴界面。
2. 印记卷放入**想复制出的目标物**；方程卷放入**想被转化的原料**。
3. 若存在对应炼金规则，物品被卷轴读取并消耗，卷轴生成槽位图。

激活后的卷轴会显示：炼金等级、源质槽位、配方要求的 EP 范围，以及以“稳定”“不稳”“濒临崩解”等词表示的熵增程度。

## 未知槽位与发现

新生成的部分槽位可能是未知的。你可以把一次失败当作勘探：

1. 在未知位置投入一种试探源质。
2. 反应后观察返还源质的状态，推断它与槽位记录源质的关系。
3. 若两者相同，它们会湮灭，卷轴会永久记录并显示该位置的正确源质。
4. 逐步揭示全部位置，下一次按记录投料即可复现。

::: warning
卷轴槽位是**有序**的。同一批源质换了位置，就会与不同的记录发生反应。截图、抄写或让卷轴保留已揭示标记都比“凭感觉扔进去”可靠。
:::

## 等级代表什么

炼金等级就是这张卷轴需要的源质槽位数。高价值目标通常等级更高，也意味着更多试探、更多 EP 波动与更多卷轴损耗。
