## 卷轴体系总览

嬗变卷轴是炼金术士的核心工具，用于**炼金复制**（凭空复制物品）和**炼金转化**（将一种物品转化为另一种）。两种卷轴各有五个等级变体：

| 变体   | 过期规则 | 耐久度 | 说明 |
|------|---|---|---|
| 嬗变卷轴 | 每天 12:00 | 32 | 入门级，适合短期任务 |
| 地相卷轴 | 每天 12:00 | 64 | 中级，更多耐久 |
| 月相卷轴 | 每 8 天（新月 0:00） | 128 | 高级，月相周期过期 |
| 太相卷轴 | 永不过期 | 256 | 不突变，适合长期保存 |
| 无相卷轴 | 永不过期 | 不可损坏 | 终极卷轴，永不损坏 |

- **嬗变印记卷**（Sigil Scroll）用于炼金复制。
- **嬗变方程卷**（Equation Scroll）用于炼金转化。

## 印记卷：炼金复制

### 原理

炼金复制将一件物品视为"模板"，通过源质反应在炼金锅中将其**复制出一份新的**。这需要存在对应的**炼金复制配方**。

### 激活

1. 手持印记卷右击，打开卷轴界面。
2. 在右侧槽位放入**你想复制的目标物品**。
3. 如果该物品存在炼金复制配方，卷轴读取配方信息并激活。

激活后：
- 左侧显示配方另一端——即需要在炼金锅中作为转化输入的物品（通常与目标物品相同或为基材）。
- 界面下方显示源质圆环，展示需要的源质数量。
- 卷轴获得一组随机的炼金槽位（数量 = 配方等级范围内的随机值）。

### 在炼金锅中使用

- 催化剂槽：激活后的印记卷轴。
- 转化输入槽：与卷轴激活时放入的物品类型**完全一致**的物品。
- 源质输入槽：按 HUD 显示的六边形槽位图投入对应源质。

## 方程卷：炼金转化

### 原理

炼金转化将一种物品转化为另一种物品。这需要存在对应的**炼金转化配方**。

### 激活

1. 手持方程卷右击，打开卷轴界面。
2. 在左侧槽位放入**你想被转化的原料物品**。
3. 如果该物品存在炼金转化配方，卷轴读取配方信息并激活。

激活后：
- 右侧显示转化后的产物预览。
- 等级以**炼金输出物品**为准计算。

### 在炼金锅中使用

- 催化剂槽：激活后的方程卷轴。
- 转化输入槽：与卷轴激活时放入的原料**完全一致**的物品。
- 源质输入槽：按 HUD 显示的六边形槽位图投入对应源质。


::: warning 注意
槽位数量越多，需要投入的源质越多，全湮灭的难度也越大。高等级配方产出的物品通常要求更多的槽位。
:::

## 卷轴过期与突变

### 过期机制

已激活的卷轴在玩家物品栏中随时间流逝而**过期**。过期时间由卷轴的 `ExpireInfo` 决定：

- **DEFAULT**：每 24000 tick（1 个游戏日）过期一次，偏移 6000 tick（约正午 12:00）。
- **LUNAR**：每 192000 tick（8 个游戏日）过期一次，偏移 114000 tick（对应新月 0:00）。
- **太阳/虚空卷轴**：无过期组件，永不过期。

卷轴放在炼金锅中且锅空闲时也会检查过期。

### 突变效果

卷轴过期后触发**突变**：

1. 槽位目标源质可能被随机替换。突变概率 = `1 - 0.2^过期次数`。多次连续过期会使突变概率急剧升高。
2. 所有槽位的源质显示被隐藏（重新变为 `?`）。
3. 魔法数字被重置，方向槽位的箭头可能改变方向。

::: tip 提示
突变后所有源质再次变为未知，你需要重新通过反应试出每个槽位需要的源质。因此，拿到卷轴后应尽快使用，或使用太阳/虚空卷轴避免突变。
:::

## 熵值与卷轴耐久

### 熵值累积

每次槽位反应时，根据源质关系累加熵值：

| 关系 | 熵值增加 |
|---|---|
| 双倍克制 / 被双倍克制 | +2 |
| 普通克制 / 被克制 / 互克 / 共生 | +1 |
| 中性 | +0 |
| 相同（触发损毁） | +0 |

### 耐久消耗

槽位触发损毁时消耗卷轴耐久：**`1 + 当前熵值 / 4`**（整数除法）。

熵值越高，后续损毁对卷轴耐久造成的伤害越大。高熵值意味着你需要更多耐久度来完成全部槽位的湮灭。
