# 高阶炼金

炼金等级大于 8 时进入高级炼金。你必须使用**地相或更高阶卷轴**，槽位图中还会出现会改变邻居、顺序乃至槽位自身的特殊槽位。

## 与低阶炼金的差别

- 槽位仍按玩家投入顺序逐个结算。
- 特殊槽位可能在自身结算时触发，也可能延迟到全部基础反应结束后触发。
- 部分特殊槽位会伪装成普通槽位，直到该位置发生湮灭才显露。
- 等级越高，特殊槽位和隐藏槽位出现的概率越大。

1. [特殊槽位图鉴](01_special_slots)

::: warning
第一次遇到陌生特殊槽位时，先悬停阅读 HUD 描述，不要立即填满整张卷轴。高级炼金的“顺序”不仅决定源质和谁反应，也决定特殊效果能看见哪些相邻产物。
:::

> 普通炼金考验配方；高级炼金考验因果。你不只要知道每个符号做什么，还要知道它何时做。
