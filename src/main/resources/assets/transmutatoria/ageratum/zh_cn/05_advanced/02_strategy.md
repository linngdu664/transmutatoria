# 高阶布局策略

## 先判断触发时机

把特殊槽分成两组：

- **即时槽**：劣化、活化、反转、扩散、抑制、清理、还原、共振、活动、交换。它们在对应源质从输入移到输出时触发。
- **延迟槽**：自旋、不稳，以及部分槽位安排的延迟交换，在所有基础反应之后触发。

因此投料顺序也是结算顺序。一个早结算的扩散槽看不见尚未产生的邻居；一个晚结算的清理槽却可能扫掉已经完成的相邻结果。

## 五步规划法

1. **标出危险槽**：清理、扩散、共振、反转。
2. **标出方向**：活动与交换箭头指向哪里。
3. **确定先后**：哪些效果需要邻居已经存在，哪些需要邻居仍为空。
4. **预测 EP**：先算基础关系，再加特殊槽额外变化和扩散倍数。
5. **预留状态边界**：避免 +2 源质再活化或 -1 源质再劣化。

## 实战原则

- 用普通状态源质做第一次侦察，容错最高。
- 邻近清理槽的位置，尽量安排会湮灭或不劣化的源质。
- 需要共振增益时，让邻居先结算；想避免共振时，让它尽早结算。
- 扩散槽周围每多一个已生成邻居，都可能额外放大 EP。
- 日相卷轴不会重排，适合给复杂布局做长期笔记。

::: danger
高阶反应开始后，临时发现 EP 会越界已经太晚。把每个槽的基础 EP、特殊修正和扩散次数写成一列再求和，是终局炼金最稳妥的做法。
:::

## 失败后的读取顺序

先看哪些槽湮灭，确认配方知识；再看返还源质的最终状态，判断特殊槽影响；最后核对 EP。不要仅凭最终金属颜色反推基础关系，因为扩散、共振和反转都可能二次改写它。
