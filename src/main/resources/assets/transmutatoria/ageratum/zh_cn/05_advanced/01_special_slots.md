# 特殊槽位图鉴

## 直接改变本槽

| 槽位 | 效果 | 玩家要点 |
|---|---|---|
| 劣化 | 本槽源质额外 -1；锅 EP 额外 +1 | 给状态留下降空间 |
| 活化 | 本槽源质额外 +1；锅 EP 额外 -1 | 黄化源质可能越界成红石 |
| 反转 | 本槽状态变化与 EP 变化取反 | 原本安全的关系可能反向 |
| 还原 | 本槽湮灭后仍保留源质 | “同种”不再意味着清空产物 |
| 共振 | 把相邻已有产物的状态总和累加给自己 | 结算早晚影响结果 |

## 改变相邻槽

| 槽位 | 效果 | 玩家要点 |
|---|---|---|
| 扩散 | 把本槽活化/劣化效果传给所有相邻已有产物，并重复结算对应 EP | 邻居越多，EP 放大越明显 |
| 抑制 | 本槽未湮灭时，标记相邻空槽；之后放入的源质状态保持不变 | 先结算才有机会抑制后续槽 |
| 清理 | 本槽发生劣化时，清空相邻已有产物；不视为湮灭 | 珍贵已揭示结果要避开 |
| 活动 | 本槽未湮灭时，沿箭头再与相邻已有产物反应；不影响锅 EP | 箭头目标必须先有产物 |

## 改变位置或槽位本身

| 槽位 | 效果 | 玩家要点 |
|---|---|---|
| 交换 | 本槽反应后，与箭头所指的非空邻居交换产物 | 最终物品位置会变化 |
| 自旋 | 全部反应后，让相邻非空产物顺时针轮转 | 属于延迟效果 |
| 不稳 | 本槽发生活化或劣化时，全部反应后随机与另一槽交换类型 | 下一轮布局可能不同 |

箭头方向由卷轴的魔法数与槽位位置决定；卷轴重排时方向也可能变化。

::: info
输出中的残渣和红石不被视为源质，因此需要“相邻已有源质”的效果通常不会把它们当成可反应对象。
:::
