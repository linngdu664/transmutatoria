# 伟大工程

“伟大工程”不是一张终局配方，而是一条必须亲自维持的转化链：准备三种基质，令原初质料依次经历黑化、白化、黄化、赤化，最后在火中固定为贤者之石。

## 路线总览

1. [三基质](01_matrices)：盐化 → 汞化 → 硫化，为高阶卷轴提供稳定根基。
2. [四阶段](02_four_stages)：原初质料 → 黑化质料 → 白化质料 → 黄化质料 → 赤化质料。
3. [贤者之石](03_philosophers_stone)：烧炼赤化质料，完成大工。
4. [炼金宇宙观](04_cosmology)：理解“上下同构”、原初质料与混沌的含义。

## 开始前的准备

- 至少一口专门控制 EP 的嬗变之锅。
- 四种状态的储物盒，避免源质库存混乱。
- 地相及以上方程卷；后期建议日相卷。
- 翠绿石板与充足普通源质，用于规划 EP。
- 把每一阶段所需区间写在物品旁或箱子告示牌上。

::: danger
赤化要求 EP 位于 +49 至 +50，离锅的毁坏边界只有一步。先验证序列，再把 EP 推到目标区间；不要在 +50 继续做任何会令 EP 上升的试探。
:::

> 分解不是失败，腐败也不是终点。黑化让旧形死去，白化洗去残余，黄化唤醒新性，赤化才使上下合一。
