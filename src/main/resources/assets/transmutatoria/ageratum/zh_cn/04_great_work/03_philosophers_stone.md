# 贤者之石

把**红化源质**放入熔炉烧炼，得到贤者之石。火在这里不再负责分解，而是把已经完成的统一固定下来。

## 随身效果

- 主手或副手持有时，持续提供生命恢复与饱和。
- 快捷栏中存在未冷却的贤者之石时，致命伤会被抵消并留下 1 点生命；随后贤者之石进入冷却。
- 对僵尸村民右击，可直接完成净化。

## 锅边用途：校准 EP

手持贤者之石右击嬗变之锅，每次令 EP 向 0 移动 1 点：

- 负数时 +1。
- 正数时 -1。
- 0 时不变。

这让精确的 EP 要求从“解一道关系题”变为“可逐点校准”，尤其适合制作硫化基质。

## 混沌分解

把贤者之石作为催化剂，再投入一个存在炼金复制规则的物品。物品会被分解为随机普通源质，数量由其炼金等级决定。

混沌分解适合回收高价值物品、补充大批基础源质；它不能保证产出指定种类，也不会直接给出黑化、白化或黄化源质。

::: tip
贤者之石作为催化剂放在锅内时，无法同时拿在手里调 EP。先校准锅，再投入贤者之石与待分解物。
:::

> 贤者之石不是让代价消失，而是让炼金术士终于能选择代价的形式。
