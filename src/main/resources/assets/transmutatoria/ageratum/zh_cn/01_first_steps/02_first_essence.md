# 第一次分解

你将进行最简单、也最重要的反应：**嬗变分解**。

## 操作

1. 确认嬗变之锅内至少有 20 mB 水。
2. 先投入 **末影之眼**。它会进入催化剂位置。
3. 再投入 **1 枚嬗变结晶**。
4. 等待反应完成。
5. 空手右击锅，取出产物。

结晶会随机分解成十二种普通状态源质之一。前十一种各有约 9% 的机会出现；稀有的**无相源金**概率约为 1%。

::: tip
末影之眼是催化剂，不会随这次分解消失。想中止尚未完成的布置，潜行并空手右击锅，按“源质 → 原料 → 催化剂”的顺序取回内容物。
:::

## 先别急着评价产物

看似随机的金属并没有优劣之分。蚀日金善于压制劫风，渊海钢能牵动星火，鸩林锡与泣月银相生。你的第一枚源质，是你进入关系网络的起点。

接下来建议连续分解数次，凑出至少两种不同源质，再阅读[锅边操作](03_controls)与[十二源质](../02_essence/index)。
