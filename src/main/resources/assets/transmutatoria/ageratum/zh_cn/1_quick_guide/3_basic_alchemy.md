# 基础炼金

*分土于火，萃精于糙，谨慎从事；*

*它自地面升到天空，复下降至地，承纳上界与下界之力；*

## 概述

前两章你积累了源质金属、理解了相生相克。本章将实际操作炼金锅，用两种反应改变和合成源质金属，并引入**极性**这一属性。

## 炼金锅基本操作

第一章你已经使用过炼金锅了，对它的 HUD 和基本操作有了初步印象——加水、丢入催化剂和材料、等待反应、取出产物。

本章的两个反应操作方式与炼金分解相同，区别在于：投入嬗变结晶或源质金属作为催化剂后，HUD 中部会显示对应的**源质槽位图**。

这里额外补充两个实用操作：

- 手持储物盒对着炼金锅右键，可以将当前选中的源质直接投入源质槽位。**滚轮**切换储物盒当前选中的源质。
- 在反应开始前或反应途中 **Shift + 空手右键**炼金锅，可以取出材料和催化剂。

## 源质反应

**催化剂：嬗变结晶**

这是最基础的反应类型——让两种源质金属直接发生反应，根据彼此的关系各自改变状态。

1. 将**嬗变结晶**丢入催化剂槽。
2. 将两个**源质金属**丢入源质槽位。
3. 反应自动开始，快速完成。

两个金属依据相生相克关系同时变化：

| 两者关系 | 变化 |
|---------|------|
| A 克制 B | A +1，B -1 |
| A 双倍克制 B | A +2，B -2 |
| 共生 | 双方 +1 |
| 互相克制 | 双方 -1 |
| 中性 | 双方不变 |

反应结束后，右键炼金锅取出变化后的两个金属。

**实例**：幻魂汞克制蚀日金。黄化幻魂汞与黑化蚀日金反应，结果为：黄化幻魂汞 **-1** → 白化幻魂汞；黑化蚀日金 **+1** → 常态蚀日金。

## 源质融合

**催化剂：任意源质金属**

用催化金属的克制关系，聚合其所有克制对象，合成催化金属本身。

1. 将一种**源质金属**丢入催化剂槽。
2. 将催化金属**所有克制和双倍克制**的金属丢入源质槽（必须全部覆盖）。
3. 反应自动开始，快速完成。输出槽产出催化金属，所有输入源质被消耗。

不同金属需要的融合材料数量不同——克制面越广的金属需要的种类越多。无相源金克制全部 11 种金属，合成它需要投入全部。

## 极性

极性是炼金锅的一项核心属性，范围 **-50 到 50**，初始为 0。HUD 右上角表盘的**外侧指针**指示锅的当前极性。

源质反应会改变锅的极性：

| 关系 | 极性变化 |
|------|---------|
| 共生 | +2 |
| 互相克制 | -2 |
| 其他关系 | 0 |

源质融合不改变极性。

::: warning 注意
极性超出范围时，炼金锅会被破坏：**极性 > 50 变为红石块**，**极性 < -50 变为残渣块**，锅内所有物品掉落。
:::
