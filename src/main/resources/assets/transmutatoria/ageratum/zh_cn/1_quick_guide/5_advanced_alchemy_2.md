# 进阶炼金 · 下

*故我获称三重伟大的赫尔墨斯，因我拥有关于整个宇宙的三个部分的智慧；*

*关于太阳运行之工的我言已毕；*

## 概述

上一章你完成了第一次炼金合成。本章将深入卷轴炼金的细节——为什么有时湮灭消耗的耐久特别多？为什么明明全部湮灭了却没有产出？卷轴过期后会发生什么？以及当配方等级较高时，你将遇到的特殊槽位。

## 熵值

每次槽位反应，如果投入的源质与目标源质的关系不是“相同”或“中性”，就会累积卷轴的**熵值**：

| 关系 | 熵值增加 |
|------|---------|
| 双倍克制 / 被双倍克制 | +2 |
| 普通克制 / 被克制 / 互相克制 / 共生 | +1 |
| 中性 / 相同（湮灭） | +0 |

卷轴的熵值越高，后续湮灭对卷轴的耐久消耗越大。这意味着暴力穷举——把每种源质都试一遍——会付出沉重代价。聪明的炼金术士会利用相生相克规则，从反应结果中推断目标源质的身份。

卷轴界面左下角会显示当前的稳定度。熵值越高，描述越倾向于不稳。

## 一次性配方

部分炼金配方是一次性的——成功产出物品后，无论卷轴还有多少耐久，催化剂槽中的卷轴都会化为乌有。

## 极性窗口

第三章介绍了极性。在卷轴炼金中，极性有了新的意义：每个炼金配方对极性有要求范围（如 -10~10）。**即使所有槽位全部湮灭，如果锅的当前极性不在配方要求的范围内，也不会产出**。

投入激活卷轴后，HUD 右上角表盘会显示极性窗口：**两个内侧指针**标记了配方要求的极性范围，**外侧指针**表示锅的当前极性。外侧指针落在两个内侧指针之间时，极性才满足条件。

卷轴反应中，不匹配的槽位反应也会改变极性（目标源质“虚”的变化体现为极性改变）。因此在开始合成前，先检查外侧指针是否在窗口内。如果偏差较大，可以用第三章的源质反应（共生/互克）预先调整极性。

## 过期与突变

嬗变、地相、月相卷轴会随时间**过期**（日相和无相永不过期）。过期规则见上一章的等级表。投入卷轴后，HUD 右上角表盘的**沙漏**会显示过期倒计时。

卷轴过期后触发**突变**：

- 所有槽位的目标源质重新隐藏（变回 `?`）。
- 部分槽位的目标源质可能被随机替换。

突变意味着你需要重新探索正确的源质。因此，激活卷轴后应尽快使用。对于耗时较长的复杂合成，建议使用月相或更高等级的卷轴。

## 高级槽位

前面的章节中，所有槽位的行为都遵循基础的相生相克规则。但当配方等级**超过 8** 时，生成的槽位有概率为**特殊类型**——它们会改变反应的行为。

等级越高，特殊槽位出现概率越大、数量越多。部分特殊槽位在生成后还会**隐藏类型**（外观和普通槽位无异），只有触发湮灭后才会揭示其真实类型。

目前有 12 种特殊槽位，例如：

- **反转槽**：将克制关系颠倒（克制变被克制，共生变互克）。
- **扩散槽**：将本槽位的状态变化传播给相邻槽位。
- **抑制槽**：若未湮灭，则锁死相邻空槽位，使它们的后续非湮灭反应无效。
- **还原槽**：湮灭时不消耗投入的源质。

完整的槽位类型及其行为说明见下方[附录·槽位类型](#附录槽位类型)。

当存在特殊槽位时，**槽位投料顺序至关重要**——反应逐槽位结算，结算顺序就是槽位被填入源质的先后顺序。先填槽 1 再填槽 2，与先填 2 再填 1，结果可能完全不同。

例如抑制槽：若先填抑制槽，它先反应，相邻空槽位被锁住，后续填入的源质非湮灭反应无效；若先填相邻槽，它们已先结算完毕，后续抑制槽无法再干扰。

## 附录·槽位类型

以下说明中，**本槽**指正在结算的槽位，**相邻槽位**指六边形周围最多六个槽位；**已有产物**指本次反应中已经结算并留在输出侧的源质。

有箭头的槽位会沿箭头方向影响一个相邻槽位。若槽位类型尚未揭示，箭头也会隐藏；当该槽位湮灭并揭示后，HUD 才会显示真实方向。

| 槽位 | 行为 |
|------|------|
| 普通槽位 | 根据源质关系进行基础反应；投入同种源质时发生湮灭，并揭示槽位。 |
| 劣化槽位 | 基础反应后，产物额外劣化 1 级，锅的极性额外增加 1 点。 |
| 活化槽位 | 基础反应后，产物额外活化 1 级，锅的极性额外降低 1 点。 |
| 反转槽位 | 反转基础反应的状态变化和极性变化。原本活化会变为劣化，原本提升极性会变为降低极性。 |
| 扩散槽位 | 本槽反应后，将本槽的活化或劣化效果传递给所有相邻、已有且未被抑制的产物；本槽及每次成功扩散都会结算一次本槽的极性变化。 |
| 抑制槽位 | 若本槽未发生湮灭，则抑制相邻的空槽位。被抑制槽位的源质状态保持不变，也不再改变锅的极性和熵值；但它仍可湮灭、被清除或被交换。 |
| 清理槽位 | 若本槽源质发生劣化，则清空所有相邻的已有产物；该清除不视为湮灭。 |
| 还原槽位 | 本槽发生湮灭时会保留投入的源质，不会将其清除。 |
| 共振槽位 | 本槽反应后，将所有相邻已有产物的状态总和累加到自身。黑化、常态、白化、黄化分别按 -1、0、1、2 计算。 |
| 活动槽位 | 若本槽未湮灭，则沿箭头与相邻的已有产物额外反应一次；同源质时双方湮灭，否则双方按关系改变状态。该额外反应不影响锅的极性。 |
| 交换槽位 | 本槽反应后，若箭头所指的相邻槽位已有产物，则与其交换产物。 |
| 自旋槽位 | 所有槽位反应结束后，将相邻的所有非空产物顺时针旋转。 |
| 不稳槽位 | 若本槽源质发生活化或劣化，则在所有槽位反应结束后，随机与另一槽位交换类型。 |
