*太阳为父，月亮为母，从风孕育，从地养护；*

*世间一切完美之源就在此处，其能力从地上获取；*

## 概述

掌握了源质基础后，你需要学会使用**嬗变卷轴**来复制和转化物品，使用**炼金术士储物盒**来管理你的源质金属收藏，以及操作**混沌炼金锅**的 HUD 界面来完成炼金合成。

## 制作卷轴

卷轴分为两种功能类型，各有五个等级变体。基础级的嬗变卷轴合成配方如下：

### 嬗变印记卷

用于**炼金复制**——凭空复制物品。

| 材料 | 说明 |
|---|---|
| 任意源质金属 | 提供炼金亲和 |
| 嬗变结晶 | 核心催化 |
| 纸 | 承载符文 |
| 金粒 | 传导炼金能量 |
| 萤石粉 | 赋予"印记"之力 |

### 嬗变方程卷

用于**炼金转化**——将一种物品转化为另一种。

| 材料 | 说明 |
|---|---|
| 任意源质金属 | 提供炼金亲和 |
| 嬗变结晶 | 核心催化 |
| 纸 | 承载符文 |
| 金粒 | 传导炼金能量 |
| 红石粉 | 赋予"转化"之力 |

两种卷轴的区别仅在于萤石粉/红石粉的选择——这决定了卷轴是"复制"还是"转化"。

### 卷轴等级变体

基础嬗变卷轴可以通过添加更多材料升级：

| 变体 | 耐久 | 过期规则 | 特点 |
|----|---|---|---|
| 嬗变 | 32 | 每天 12:00 | 入门级 |
| 地相 | 64 | 每天 12:00 | 中等耐久 |
| 月相 | 128 | 每 8 天（新月） | 长周期 |
| 日相 | 256 | 永不过期 | 适合长期保存 |
| 无相 | 不可损坏 | 永不过期 | 终极卷轴 |

::: tip 提示
高级卷轴需要更多资源，但提供了更好的耐久和更宽松的过期机制。详见[炼金复制与炼金转化](../3_replication_and_transformation/index.md)章节。
:::

## 激活卷轴

卷轴在刚制作出来时是**空白**的——需要放入一个目标物品来激活。

### 激活步骤

1. **手持卷轴右击**，打开卷轴界面。
2. 放入目标物品：
   - **印记卷**：放入你**想复制出的目标物**（右侧槽位）。卷轴会自动查找炼金复制配方。
   - **方程卷**：放入你**想被转化的原料**（左侧槽位）。卷轴会自动查找炼金转化配方。
3. 如果该物品存在对应的炼金规则，物品被卷轴**读取并消耗**，卷轴被激活。

### 激活后的变化

- 卷轴界面显示配方另一端的物品预览。
- 界面下方显示**源质圆环**，标注需要源质的数量（即槽位数量）。
- 卷轴获得一组随机的**炼金槽位**，每个槽位有隐藏的目标源质类型。
- 刚激活后所有源质显示为 **`?`**（未知），需要在炼金锅中通过反应逐一揭示。

::: warning 注意
激活时放入的物品会被**消耗**。请确认你放入的是正确的物品——印记卷放"想复制的目标"，方程卷放"想被转化的原料"。
:::

## 炼金术士储物盒

炼金术士储物盒是管理源质金属的核心工具。它同时是**物品**和**可放置方块**。

### 功能

- 提供 **12 个槽位**，每个槽位绑定一种源质金属。
- **自动收纳**：捡起源质金属时，背包或副手中的储物盒会自动吸入匹配状态的源质。
- **快速投料**：手持储物盒对准炼金锅，可以快速将选中的源质投入锅的输入槽。
- 在炼金锅HUD上按住 `Alt` 键可以展开源质轮盘，显示当前储物盒内源质之间的简单克制关系。

### 四种状态变体

储物盒有四种状态变体，只能收纳**对应状态**的源质金属：

| 储物盒 | 状态 | 收纳范围 |
|---|----|---|
| 炼金术士储物盒（常态） | 0  | 常态源质金属 |
| 黑化炼金术士储物盒 | -1 | 黑化源质金属 |
| 白化炼金术士储物盒 | 1  | 白化源质金属 |
| 黄化炼金术士储物盒 | 2  | 黄化源质金属 |

### 方块形态

储物盒可以放置在地上作为方块使用：

- **右键方块**：打开储物盒菜单（有开盖动画和音效）。
- **Shift + 右键方块**：将方块收回为物品形态，内容保留在物品的 NBT 中。

### 自动收纳优先级

捡起源质金属时，系统会按以下顺序查找匹配的储物盒：
1. 副手
2. 背包（按格子顺序）

找到第一个匹配状态且有空间的储物盒即停止。

## 混沌炼金锅操作

混沌炼金锅是本模组的核心工作站。看向它时会显示 HUD 界面，这是所有炼金操作的信息中枢。

### HUD 布局

| 区域 | 位置 | 显示内容 |
|---|---|---|
| 竖条 | 左侧 | 上：催化剂 · 中：转化输入 · 下：转化输出 |
| 六边形区域 | 中央 | 卷轴激活后显示炼金槽位图；未激活时为空 |
| 沙漏圆盘 | 右上 | 反应进度条 · 当前锅极性 · 卷轴配方要求的极性范围 |
| 金属管 | 最左侧 | 卷轴当前耐久度（紫色部分 = 预计消耗） |

### 催化剂类型与槽位显示

HUD 中央的源质槽位图根据催化剂类型变化：

| 催化剂 | 显示内容 |
|---|---|
| 嬗变结晶 | 2 个源质反应槽 |
| 任意源质金属 | 该金属需要的克制/双倍克制集合槽位 |
| 激活的卷轴 | 六边形连通的炼金槽位图（含方向箭头、类型标记） |
| 贤者之石 / 末影之眼 | 无源质槽位（使用转化输入槽） |

### 源质轮盘

手持炼金术士储物盒看向炼金锅时，屏幕上方显示**12 槽环形轮盘**：

- **中间**是当前选中的源质。
- **旋转鼠标滚轮**切换选中的源质。
- 按住 `Alt` 展开克制关系线。

### 槽位选择与投料

- 输入源质槽位上有一个**选择环**，指示当前操作的槽位。
- **按住 Shift + 滚轮**：切换选择槽位。
- **右键**：将储物盒当前选中的源质投入选中槽位。
- 也可以直接将源质金属物品**丢到锅上**，它会自动进入匹配的槽位。

### 反应触发条件

反应开始需要满足以下全部条件：

1. 锅内有足够的水（≥ 20 mB）。
2. 当前没有正在进行的反应（进度条为 0）。
3. 没有未取走的产物。
4. 催化剂已放入。
5. 根据催化剂类型，转化输入和/或源质输入已就位。

满足条件后反应自动开始，右侧沙漏圆盘显示进度。

## 炼金合成

激活卷轴后，将其投入混沌炼金锅的催化剂栏位。HUD 会显示卷轴的源质槽位图——这是一组六边形连通的槽位，每个槽位标记着它需要的源质金属类型（初始为未知，每解锁源质会显示）。

### 执行合成的步骤

1. **投入卷轴**：将已激活的卷轴丢入炼金锅，它进入催化剂槽位。
2. **放入转化原料**：印记卷轴需要放入**你想要复制的那个物品**；方程卷轴需要放入**你想被转化的那个物品**。放入的物品必须与卷轴激活时所用的物品类型一致。
3. **填充源质槽位**：从炼金术士储物盒中取出源质金属，依次填入 HUD 显示的每个源质输入槽。按住 `Shift` 滚动鼠标滚轮可切换当前操作的槽位；右键即可将储物盒当前选中的源质投入该槽位。
4. **等待反应完成**：投入的源质数量满足卷轴要求后，反应自动开始。沙漏圆盘显示当前反应进度。
5. **收取产物**：反应结束后，产物出现在转化输出槽。右键炼金锅即可取出。

### 反应的内部过程

反应按你投放源质的顺序逐个执行槽位：

- 每个槽位根据**投入的源质**与**该槽位目标源质**的关系计算状态变化。基础规则与源质反应相同（克制方活化、被克制方劣化），但每个槽位的**特殊类型**会改变这一规则。
- 反应过程中，输入源质会被消耗并产生对应状态变化的输出源质，写入输出槽。
- 如果某个槽位投入的源质**与目标源质相同**，该槽位**触发损毁**（湮灭），并揭示该槽位的类型和目标源质。每次触发损毁都会消耗卷轴的耐久度，消耗量 = `1 + 当前熵值 / 4`。
- **只有当所有槽位全部触发损毁**，且锅的极性在卷轴配方要求的极性范围内，且转化输入仍然匹配卷轴要求时，转化输出槽才会产出最终物品。
- 无论是否成功产出，反应结束后所有输入源质都会被清空。

### 一次性与可复用卷轴

卷轴分为**一次性**和**可复用**两种：

- **一次性卷轴**（`one_time = true`）：成功产出物品后，催化剂槽位的卷轴会被消耗。
- **可复用卷轴**：即使成功产出，卷轴仍保留在催化剂槽位中，可以再次放入转化原料和源质进行下一轮合成。

### 极性管理

每次炼金合成反应都会改变锅的极性。若极性超出 ±50：

- 极性 > 50：锅变为**红石块**，所有物品掉落。
- 极性 < -50：锅变为**炼金渣块**，所有物品掉落。

手持**贤者之石**右键炼金锅，可以将极性向 0 回调 1 点，用于微调。

### 观察 HUD 信息

- **左边竖条**：上中下分别显示催化剂、转化输入、转化输出。
- **中间六边形区域**：显示卷轴的源质槽位图。激活但未揭示的槽位显示金色边框，已揭示的显示源质图标和类型标记。
- **右上沙漏圆盘**：显示当前反应进度。外圈指示极性状态。
- **最左侧金属管**：显示卷轴当前耐久度和熵值累积。紫色部分代表合成整件物品预计消耗的耐久。

::: tip 提示
如果卷轴的源质槽位图中有方向箭头（如活动槽、交换槽），箭头的实际方向由卷轴的魔法数字决定。每次卷轴突变或重新激活时方向可能变化。详见[高级炼金](../4_essence_conversion_and_reaction/index.md)章节。
:::
